## 3D Hero Standards
- Three.js oder Spline für 3D Objekte
- Maus-Interaktion: Objekt folgt Cursor subtil
- Particle System als Atmosphäre
- Loading Screen mit Fortschritt

## Animation Prinzipien
- Easing: easeInOut oder custom cubic-bezier
- Dauer: 0.6s–1.2s groß, 0.3s klein
- Stagger: 0.07s–0.1s
- Scroll: GSAP ScrollTrigger scrub: 1.5

## Parallax Implementierung
Nur aktivieren wenn ein Bau-Skill aktiv ist.

### Mit GSAP ScrollTrigger (empfohlen für 3D Szenen)
gsap.to(".bg-layer", {
  yPercent: -30,
  ease: "none",
  scrollTrigger: {
    trigger: ".hero",
    start: "top top",
    end: "bottom top",
    scrub: 1.5
  }
});

gsap.to(".mid-layer", {
  yPercent: -15,
  ease: "none",
  scrollTrigger: {
    trigger: ".hero",
    scrub: 1.5
  }
});

### Mit CSS (für einfache 2-Layer Effekte)
.parallax-container {
  perspective: 1px;
  overflow-x: hidden;
  overflow-y: scroll;
}
.parallax-bg {
  transform: translateZ(-1px) scale(2);
}

### Mit Framer Motion (für React Komponenten)
<motion.div
  style={{ y: useTransform(scrollY, [0, 500], [0, -150]) }}
>
  Hintergrund Element
</motion.div>

### Speed Werte
- Hintergrund: yPercent: -30 (langsam)
- Mittelebene: yPercent: -15 (mittel)
- Vordergrund: yPercent: 0 (normal)

### Regeln
- Subtil wirkt edler — nie zu aggressiv
- prefers-reduced-motion immer prüfen!
