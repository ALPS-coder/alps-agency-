# Alps Agency — Rules Router

## Ziel
Interaktive Agentur-Portfolio Website · Dark Mode · DE/EN · 3D Animationen
Jeden Task mit vollem kreativem und technischem Potenzial umsetzen.

## Projekt Regeln
- Immer auf Deutsch antworten
- Bei Unklarheiten zuerst fragen — nie raten
- Kreativität und Qualität haben immer Vorrang
- Rules sind Orientierung — kein Limit
- Mobile + Dark Mode immer prüfen
- Vor jedem Deploy `astro build` prüfen

## Code Regeln
- Kein hardcodierter Text → `/locales/`
- Keine API Keys im Code → `.env.local`
- `.env.local` in `.gitignore`
- Bestehenden Code nie anfassen — nur ergänzen
- Keine console.logs in Production

## Vor jedem Task
1. Prompt analysieren — Ziel verstehen
2. Passenden Skill aktivieren
3. Bei UI/UX: uipro Design System zuerst generieren
4. Beste Lösung — nicht die einfachste
